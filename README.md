# Detection Rules

Detection engineering portfolio. I'm a security architect with 20+ years across 
infrastructure and Cisco security, transitioning into hands-on detection 
engineering and AI/agent security. This repo tracks what I build as I learn.

## Focus areas

- **Cloud detection** — AWS, Azure (CloudTrail, Activity Logs, IAM patterns)
- **Kubernetes & containers** — Falco rules, k8s audit log detections
- **Endpoint & identity** — Windows event logs, sysmon, identity provider abuse
- **AI/agent security** — emerging detection patterns for LLM and agent abuse

## Structure

- `/sigma` — Sigma rules (vendor-agnostic)
- `/kql` — Microsoft Sentinel / Defender KQL queries
- `/spl` — Splunk SPL queries
- `/falco` — Falco rules for Kubernetes runtime detection
- `/python` — Automation scripts (enrichment, SOAR-style workflows)
- `/notes` — Learning notes, threat models, writeups
